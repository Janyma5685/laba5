Главная

Главная страница — основная страница сайта, с которой пользователь начинает знакомство с ресурсом.

Этот сайт создан для лабораторной работы №5. В нём используется блочная компоновка из прошлой работы, а также добавлены элементы JavaScript: диалоговые окна, работа с формами, открытие дочерних окон и web-калькулятор.

В навигационном меню доступны обычные страницы сайта, секретная страница с проверкой пароля, сумматор и web-калькулятор. Ссылка «Контакты» на главной странице открывается в новом окне заданного размера.

Оформление сайта вынесено в файл styles/style.css, а повторно используемые сценарии JavaScript — в файл script.js.
